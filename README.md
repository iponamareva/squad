# squad

Данный проект реализует нейросеть, которая по заданному пользователем контексту и вопросу ищет ответ на этот вопрос в контексте. Нейросеть является адаптированной версией нейросети, описанной в следующей статье: https://arxiv.org/pdf/1704.00051.pdf

## Описание



Для того, чтобы иметь возможность запустить скрипты к этому проекту, необходимо иметь установленный пакет tensorflow. Его установка может оказаться нетривиальной задачей, однако все необходимые инструкции по установке можно найти на официальном сайте Tensorflow:

https://www.tensorflow.org/install/
